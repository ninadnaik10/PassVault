# PassVault
A password manager built using Java Swing and SQLite

### Setup
- Download the zip file from the [releases](https://github.com/ninadnaik10/passvault/releases) page.
- Unzip it and run the jar file.

For developer setup, check out [CONTRIBUTING.md](https://github.com/ninadnaik10/passvault/blob/main/CONTRIBUTING.md).
